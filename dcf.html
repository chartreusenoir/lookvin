公式驱动区（所有其他Sheet）

财务预测：

营收 = 上一年营收 × (1 + 假设!增长率)
EBIT = 营收 × 假设!EBIT利润率
NOPAT = EBIT × (1 - 假设!税率)
FCFF = NOPAT + D&A - Capex - ΔNWC

WACC：

Ke = Rf + β × ERP
WACC = Ke × (1-D/V) + Kd×(1-t) × D/V

DCF估值：

终值 TV = FCFF₁₀ × (1+g) / (WACC - g)
EV = Σ[FCFFₙ/(1+WACC)ⁿ] + TV/(1+WACC)¹⁰
每股价值 = (EV - 净负债) / 股本

只需修改"假设参数"Sheet中的任意一个数字，整个模型所有结果自动联动更新。

图表问题修复

根因：原来用横向单行 Reference 调用 add_data()，openpyxl 把每列都当成独立 Series，10年数据产生10个 Series，但只给第0个手动设了标题，其余9个是 None。

修复：在图表Sheet右侧建一个竖排辅助数据区（年份/FCFF/PV各一列），用 titles_from_data=True 让标题自动从第一行读取，数据只生成 1个 Series，图例完全正确。同时设置 legend=None（单系列图无需图例）。

敏感性分析公式解释

以基准格为例，完整公式拆解：

=IF(G4<=A7, "N/A",
  (
    财务预测!C10/(1+G4)^1  +  ← Y1 FCFF 折现到今天
    财务预测!D10/(1+G4)^2  +  ← Y2 FCFF 折现
    ...
    财务预测!L10/(1+G4)^10 +  ← Y10 FCFF 折现
    (财务预测!L10*(1+A7)/(G4-A7))  ← 终值（Gordon公式）
      / (1+G4)^10              ← 终值折现至今
    + 假设参数!B34             ← 加回净现金（负净负债）
  ) / (假设参数!B35 * 100)     ← ÷ 股数（百万股）= 元/股
)
公式部分含义
G4本列 WACC（列标题行公式 = WACC基准 ± offset，动态计算）
A7本行 g（行标题列公式 = g基准 ± offset，动态计算）
IF(G4<=A7,"N/A",…)防止 WACC≤g 时 Gordon 分母为零或负数
财务预测!C10…L10直接引用财务预测Sheet的FCFF行，不重复计算
(FCFF₁₀×(1+g)/(WACC-g))/(1+WACC)¹⁰Gordon终值折现至今
+假设参数!B34加回茅台净现金约1960亿（负净负债）
/(股本×100)百万元÷百万股=元/股

每个格子都是完全独立的封闭公式，改变假设参数任意数字，整行整列自动重算。矩阵覆盖 WACC ±200bp、g ±150bp 的完整区间。
